# Quick Start Guide

Get up and running with RepoRadar in 5 minutes!

## Your First Analysis

### 1. Sign In

Visit `http://localhost:5000` and sign in with your account. New users will see an interactive onboarding tour.

### 2. Analyze a Repository

**Option A: Quick Analysis**
1. Paste a GitHub repository URL in the search bar
2. Click "Analyze"
3. Wait for AI-powered analysis (usually 10-30 seconds)

**Option B: Advanced Search**
1. Click "Discover" → "Advanced Search"
2. Set filters (language, stars, topics, etc.)
3. Browse results and click "Analyze" on any repository

### 3. View Results

Your analysis includes 5 key metrics:

- **Originality** (0-100): How unique and innovative the project is
- **Completeness** (0-100): Documentation, tests, and project maturity
- **Marketability** (0-100): Potential for adoption and growth
- **Monetization** (0-100): Revenue generation potential
- **Usefulness** (0-100): Practical value and problem-solving capability

Each metric includes:
- Score with visual indicator
- Detailed AI explanation
- Actionable recommendations

## Key Features

### Batch Analysis

Analyze multiple repositories at once:

1. Navigate to "Discover" → "Batch Analysis"
2. Add repository URLs (up to 3 for free, unlimited for Pro)
3. Click "Start Batch Analysis"
4. Track progress in real-time
5. Export results as PDF or CSV

### Find Similar Repositories

Discover related projects:

1. Open any analyzed repository
2. Click "Find Similar"
3. Choose similarity type:
   - By metrics (similar scores)
   - By functionality (similar purpose)
4. Browse recommendations

### Collections & Bookmarks (Pro)

Organize your repositories:

1. Click "Workspace" → "Collections"
2. Create a new collection
3. Add repositories from analyses
4. Access bookmarks for quick reference

### Compare Repositories

Side-by-side comparison:

1. Navigate to "Discover" → "Compare"
2. Select 2-4 repositories
3. View metric comparisons
4. Export comparison report

## Tips for Best Results

### 1. Use Advanced Search Effectively

Combine filters for precise results:
```
Language: JavaScript
Stars: 1000+
Topics: react, typescript
Updated: Last 6 months
License: MIT
```

### 2. Leverage AI Recommendations

Pro users get personalized repository suggestions based on:
- Analysis history
- Saved collections
- Search patterns

### 3. Export and Share

Export analyses in multiple formats:
- **PDF**: Professional reports with charts
- **CSV**: Data for spreadsheets and analysis
- **JSON**: API integration

### 4. Monitor Trending Repositories

Stay updated with trending projects:
1. Visit "Discover Trending"
2. Filter by language or time period
3. Analyze interesting repositories
4. Add to collections

## Keyboard Shortcuts

Speed up your workflow:

- `Ctrl/Cmd + K`: Quick search
- `Ctrl/Cmd + B`: Toggle bookmarks
- `Ctrl/Cmd + N`: New analysis
- `Esc`: Close modals

## Understanding Metrics

### Originality
Evaluates innovation, unique features, and differentiation from existing solutions.

**High Score Indicators:**
- Novel approach or technology
- Unique feature set
- Creative problem-solving

### Completeness
Assesses documentation quality, test coverage, and project maturity.

**High Score Indicators:**
- Comprehensive README
- API documentation
- Test coverage >80%
- Active maintenance

### Marketability
Measures potential for adoption and community growth.

**High Score Indicators:**
- Clear value proposition
- Active community
- Good documentation
- Regular updates

### Monetization
Evaluates revenue generation potential.

**High Score Indicators:**
- Commercial viability
- Market demand
- Scalability
- Business model clarity

### Usefulness
Rates practical value and problem-solving capability.

**High Score Indicators:**
- Solves real problems
- Easy to use
- Well-maintained
- Active user base
