# RepoRadar Features

Comprehensive AI-powered analysis and discovery for GitHub repositories.

## Core Analysis Features

### [Repository Analysis](./repository-analysis.md)
AI-powered evaluation across 5 key metrics using Google Gemini 2.5 Pro.

**What you get:**
- Originality, Completeness, Marketability, Monetization, Usefulness scores (0-100)
- Detailed explanations for each metric
- 3-5 strengths with evidence
- 3-5 weaknesses with impact analysis
- 3-5 actionable recommendations
- Comprehensive summary
- Overall weighted score

**Use cases:**
- Evaluate libraries before adoption
- Assess project quality
- Make informed technology decisions
- Identify improvement opportunities

[Learn more →](./repository-analysis.md)

### [Batch Analysis](./batch-analysis.md)
Analyze multiple repositories simultaneously with real-time progress tracking.

**Capabilities:**
- 3 repositories (Free), unlimited (Pro)
- Parallel processing
- Real-time status updates
- Batch export (PDF/CSV)
- Side-by-side comparison

**Use cases:**
- Compare competing libraries
- Evaluate technology stacks
- Audit dependencies
- Competitive analysis
- Portfolio review

[Learn more →](./batch-analysis.md)

### [Similar Repositories](./similar-repositories.md)
Discover related projects using AI-powered similarity search.

**Two search methods:**
- **By Metrics**: Find repositories with similar scores
- **By Functionality**: AI-powered functional similarity

**Use cases:**
- Find alternatives
- Discover competitors
- Explore new tools
- Get inspiration
- Build tech stacks

[Learn more →](./similar-repositories.md)

## Discovery Features

### Advanced Search
Find repositories with 15+ powerful filters.

**Available filters:**
- Programming language (50+ languages)
- Star count range
- Fork count range
- Issue count range
- Last update date
- Creation date
- License type
- Topics and keywords
- Repository size
- Has wiki
- Has issues enabled
- Is archived
- Is fork
- And more...

**Features:**
- Combine multiple criteria
- Save search templates (Pro)
- Export results
- Quick analyze from results

### Discover Trending
Find popular repositories by language and timeframe.

**Options:**
- Filter by programming language
- Time periods: Today, This Week, This Month
- Sort by stars, forks, or recent activity
- Quick analyze trending projects

### Compare Repositories
Side-by-side comparison of 2-4 repositories.

**Compare:**
- All 5 metric scores
- Repository metadata
- Community metrics
- Strengths and weaknesses
- Visual charts
- Export comparison report
